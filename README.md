# Avelure Social Command Dashboard 2026

A self-contained social media management dashboard for **Avelure Med Spa**, built and maintained by the Weramp / AMP marketing team.

## Overview

This dashboard is the single source of truth for Avelure's social media rollout across Instagram, Facebook, and Linktree for the full 2026 calendar year. It is designed to be used by the social media account manager, marketing specialists, and reviewed by senior marketing leadership.

---

## What's Inside

| Section | Description |
|---|---|
| **Dashboard** | Live progress tracker with overall completion percentage across all tasks |
| **Project Tracker** | Every setup task across Instagram, Linktree, Facebook, and Content — with checkboxes |
| **Instagram** | 12-step phased optimization plan (Profile → Content System → Growth) |
| **Linktree** | 4-link setup spec with confirmed URLs and brand styling guide |
| **Facebook** | Phased page setup, cross-posting configuration, Meta Business Suite, and Pixel install |
| **Content Calendar** | Full month-by-month calendar March–December 2026 with detailed post briefs, caption direction, and Facebook cross-post indicators |
| **Highlights Spec** | Final 6 Instagram Highlight cover assignments with brand hex codes and Canva build instructions |
| **Resources & Links** | All live account links, SOPs, brand assets, SharePoint docs, and collaboration/UGC guide |

---

## How to Use

1. Open `index.html` in any modern browser — no server or build step required
2. Use the left sidebar to navigate between sections
3. Click any checkbox or post card to mark it complete — progress updates in real time
4. The Content Calendar is organized by month — click any month tab to view that week's posts
5. Each calendar post includes: post type, content brief, caption direction, and a Facebook cross-post badge where applicable
